Сюжет

Когда мир рухнул, она осталась одна. Люди борются за выживание, природа начинает обращаться в пустоши, и даже ветер перестал петь свои песни. Волчица по клички Скала — последняя из стаи, выжившая среди руин былой гармонии. Но внутри неё всё ещё теплилась искра — её детёныши.

Три хрупких жизни, которые теперь зависят только от неё. Семье нужно убежать. Скале предстоит пройти через ад — сквозь земли, где больше ничего не растёт, сквозь воспоминания о мире, который уже не вернуть.

Игра — это её путь. Путь матери, которая не сдаётся. Каждый шаг — испытание. Каждая встреча — риск. Она будет охотиться, защищать, лгать. Иногда — проигрывать.

В этом мире нет героев. Только те, кто ещё не сдался.
